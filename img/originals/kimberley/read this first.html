  • Use them in scrapbook kits.

  • Embed the fonts in PDF files, software, web pages and more. Read about embedding here.

Things you can’t do with my free fonts

  • Embed fonts in poster generators, LOLcat generators, T-Shirt generators, business card creation apps etc. Read about embedding here.

Font vendors’ own agreements

Some font vendors issue their own license agreements with my fonts. In that case, the rules of my license agreement may not apply.

Custom license

If you’re interested in OEM licensing or anything else not covered by the license agreement, you can set up a custom agreement.

Font problems

You can easy font installation guides on YouTube. If you have problems with this font, get the latest version from a font vendor. These vendors can provide customer service and some even provide an automatic font installation tool. If there’s a problem with the font, inform the vendor, they’ll contact me and I’ll fix it.

OT or TT?

This archive contains 2 font formats: OpenType(OT) & TrueType(TT). I recommend installing TrueType(TT). Install the other format if you absolutely require it.
